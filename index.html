🚀 Contribute Your Template

📝 How to Contribute

  1. Fork the Repository: Fork the main repository on GitHub
  2. Add Your Template: Create your .md file in the appropriate category folder:
    • templates/minimalistic/ - Clean and simple designs
    • templates/showcase-collections/ - Comprehensive showcase profiles
    • templates/creative-artistic/ - Creative and artistic templates
    • templates/code-focused/ - Developer and tech-focused
    • templates/data-visual/ - Stats and data-driven
    • templates/dynamic-interactive/ - Interactive content
    • templates/media-rich/ - Rich media and multimedia
    • templates/badges-icons/ - Badge and icon focused
    • templates/others/ - Unique and misc templates
  3. File Naming: Use your GitHub username as filename: your-username.md
  4. Submit PR: Create a pull request with your template
  5. Auto-Update: Once merged, the site will automatically detect and display your template!
